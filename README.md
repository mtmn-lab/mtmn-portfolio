## 2. 経営改善DXプロジェクト（マジカルショッププロジェクト

## 概要
    架空店舗「マジカルショップ」を題材に、経営改善・業務改善をテーマとした店舗DXプロジェクトを進行中。
    
    店舗経営をテーマに、売上・原価・人件費・利益データを統合し、経営状態を可視化する分析基盤を構築
    Power BI / SQL / Python を用いて、粗利分析、利益貢献分析、損益分岐分析などを実施。
    今後は在庫管理・仕入れ管理・業務改善まで含めた店舗DXプロジェクトへ拡張予定。

## 背景・課題
    店舗運営において、「なぜ利益が残らないのか」が感覚的な判断になっており、実態把握が困難な状態となっていた。
    また、スタッフの売上のみでスタッフ評価が行われており、粗利・原価率・利益貢献など、
    本質的な指標が可視化されていなかった。

## 使用技術
    - Power BI
    - SQL (PostgreSQL)
    - Python
    - DBeaver
    - Cursor

## 現在実装済み

### SQL
    - スタッフ別売上分析
    - 実仕入れベース粗利分析
    - 利益貢献額算出
    - 月次損益分析
    - 損益分岐点分析
    - Dataflow設計
    - 運用定着支援

### Power BI
    - 月次売上 / 利益ダッシュボード
    - 損益分岐点可視化
    - スタッフ別分析
    - 原価率分析

### データ整備
    - スタッフマスタ作成
    - 実績ログ整備
    - SQLビュー作成
    Pythonスクリプトを用いて、データ収集・加工処理を実施。
    AIツールも活用しながら、分析しやすいデータ構造への整備を進行中。

## 分析から得られた知見
    - 売上だけでは見えない利益構造を可視化
    - 実仕入れベース分析により、スタッフごとの利益率差を発見
    - 高売上スタッフと高利益率スタッフが必ずしも一致しないことを確認
    - 人件費をフックメタ利益貢献分析を実施

## 今後の完全予定

    - 仕入れ分析強化
