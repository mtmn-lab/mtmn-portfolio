# 🛠 Philosophy

- BIは「可視化して終わり」ではなく改善サイクルの一部
- 「全部自動化」ではなく、人間確認を含めた運用設計を重視
- 現場で実際に使われるDXを目指す
- 小規模改善を積み重ね、運用へ定着させることを重視
- 「めんどくさい」を減らし、人が本来考えるべき業務へ集中できる環境を作りたい
