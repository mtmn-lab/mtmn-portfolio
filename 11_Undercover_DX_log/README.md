# 潜入DXログ

## Purpose

現場業務を理解しながら、
「どこに改善余地があるか」
「どの技術で負荷軽減できるか」
を観察・整理する目的で記録。

業務効率化だけでなく、
情報整理・ナレッジ共有・データ活用まで含めて
改善視点を残している。

詳細な改善ログ・設計メモは個人管理環境で継続記録中
